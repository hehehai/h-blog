### 修改 PATH

为了直接在终端使用 MySQL cli，我们要配置下 PATH

先查看下当前的 PATH，在终端输入 `env`

![mysql-cli-os-env](/static/images/mac-install-mysql/mysql-cli-os-env.png)

1. 修改 `.zshrc` 文件，扩展 PATH

```shell
vi ~/.zshrc
```

我这里使用 VIM， 你也可以使用其他编辑器，例如 vscode `code ~/.zshrc`

```shell
export PATH=$PATH:/usr/local/mysql/bin
```

将 MySQL cli bin 路径添加到 env PATH 后

2. 使 PATH 配置生效

```shell
source ~/.zshrc
```

这会重载 zsh 的配置，执行我们的 PATH 扩展命令

查看 PATH, 在终端输入 `env`

![mysql-cli-os-path](/static/images/mac-install-mysql/mysql-cli-os-path.png)

3. 验证 MySQL cli

```shell
mysql --version
```

![mysql-cli-version](/static/images/mac-install-mysql/mysql-cli-version.png)
